# PreOrder Traversal

## Descripción

Este programa resuelve el problema **Tree: Preorder Traversal** de HackerRank.

El objetivo es realizar el recorrido **PreOrder** de un árbol binario. En este tipo de recorrido se visitan los nodos siguiendo el orden:

**Raíz → Subárbol izquierdo → Subárbol derecho**

La solución se implementó utilizando recursividad, recorriendo cada nodo del árbol una única vez.

## Tecnologías utilizadas

- C++
- Visual Studio Code
- Compilador g++

## Cómo compilar

Abrir una terminal en la carpeta del proyecto y ejecutar:

```bash
g++ main.cpp -o PreOrderTraversal
```

## Cómo ejecutar

En Windows:

```bash
.\PreOrderTraversal.exe
```

## Ejemplo de entrada

```
6
1
2
5
3
4
6
```

## Salida esperada

```
1 2 5 3 4 6
```
