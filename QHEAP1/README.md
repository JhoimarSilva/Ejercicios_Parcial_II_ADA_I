## Descripción

Este programa resuelve el problema **QHEAP1** de HackerRank.

El objetivo es procesar una serie de consultas sobre un conjunto de números enteros. Las operaciones que puede realizar el programa son:

- Insertar un elemento.
- Eliminar una ocurrencia de un elemento.
- Imprimir el elemento mínimo almacenado.

Para resolver el problema se utiliza un **multiset** de C++, ya que esta estructura mantiene los elementos ordenados automáticamente y permite realizar las operaciones solicitadas de forma eficiente.

## Tecnologías utilizadas

- C++
- Visual Studio Code
- Compilador g++

## Cómo compilar

Abrir una terminal en la carpeta del proyecto y ejecutar:

```bash
g++ main.cpp -o QHEAP1
```

## Cómo ejecutar

En Windows:

```bash
.\QHEAP1.exe
```
## Ejemplo de entrada

```
5
1 4
1 9
3
2 4
3
```

## Salida esperada

```
4
9
```
