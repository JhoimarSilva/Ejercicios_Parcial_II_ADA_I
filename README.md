## Descripción

Este repositorio contiene el desarrollo de la **Parte II del Parcial 2** de la asignatura **Análisis y Diseño de Algoritmos I (ADA I)**.

Como parte de la evaluación se seleccionaron y resolvieron ejercicios de HackerRank pertenecientes a las categorías propuestas por el docente.

Para cada ejercicio se realizó:

- Implementación en C++.
- Análisis de complejidad temporal y espacial.
- Pruebas de funcionamiento.
- Documentación mediante README.
- Preparación para la sustentación.

## Objetivo

Aplicar los conceptos estudiados en la asignatura **Análisis y Diseño de Algoritmos I**, implementando soluciones eficientes para diferentes problemas relacionados con ordenamiento, cadenas de caracteres, árboles, heaps y búsqueda, utilizando estructuras de datos y algoritmos apropiados para cada caso.
